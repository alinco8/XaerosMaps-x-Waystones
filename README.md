## What is this mod?

This mod connects Waystones with Xaero’s Minimap / World Map!

More specifically...

- When you place a Waystone (**Waystone only** — Sharedstone and others are not supported), it gets
  automatically added as a marker in Xaero’s Minimap / World Map.
- When you break a Waystone, the matching marker is automatically removed too.
- When you rename a Waystone, the marker name updates automatically as well.
- Fully configurable!
